# От бомжа до президента

Мобильная игра для Telegram Web App. Игрок начинает бомжом под мостом и проходит путь через шабашки, работу, образование, бизнес и политику до президентских выборов.

## Что есть в проекте

- мобильный жёлтый дизайн;
- рубли и доллары;
- здоровье, сытость, счастье и энергия;
- 10 категорий и более 40 действий;
- отдельная категория «Популярность» с доступными с начала игры способами прокачки;
- образование, репутация, связи, популярность и влияние;
- жильё, активы и пассивный доход;
- случайные события;
- карьерная лестница от бомжа до президента;
- сохранение игры в `localStorage`;
- интеграция с Telegram Web App SDK.

## Запуск локально

```bash
python -m http.server 8000
```

Затем откройте `http://localhost:8000`.

## Публикация на GitHub Pages

1. Создайте новый репозиторий.
2. Загрузите в него `index.html`, `styles.css` и `app.js`.
3. Откройте Settings → Pages.
4. Выберите Deploy from a branch, ветку `main`, папку `/root`.
5. Полученную HTTPS-ссылку укажите в BotFather через `/newapp` или в кнопке Web App.

Сохранение хранится в браузере конкретного устройства. Для одного аккаунта на разных устройствах потребуется backend и база данных.

## Быстрый запуск на Windows

1. Распакуйте архив.
2. Дважды нажмите `start_game.bat`.
3. Игра автоматически откроется в браузере по адресу `http://localhost:8000`.
4. Чтобы остановить сервер, закройте окно командной строки или нажмите `Ctrl+C`.

Для запуска нужен установленный Python 3. Если Windows не находит Python, переустановите его с включённой опцией **Add Python to PATH**.

## Запуск на Windows

1. Полностью распакуйте папку `homeless_survival_tg_app_v1_7`.
2. Дважды нажмите `start_game.bat`.
3. Игра откроется напрямую в браузере. Python и локальный сервер для обычного запуска больше не нужны.

Если нужен запуск через HTTP-сервер, используйте `start_server.bat`. Он остановит старый процесс на порту 8765, запустит сервер из правильной папки и откроет `http://127.0.0.1:8765/`.

Для остановки HTTP-сервера используйте `stop_game.bat`.

## Смерть персонажа

Если здоровье, сытость или счастье падают до 0, игра показывает предупреждение и запускает таймер на 24 игровых часа. Если показатель остаётся на нуле все 24 часа, прохождение заканчивается смертью. Любое восстановление показателя выше 0 полностью сбрасывает его таймер.
