# От бомжа до президента — v1.16

Мобильная игра для Telegram Web App. Игрок начинает бомжом и проходит путь через подработки, образование, бизнес и политику до президентских выборов.

## Основные механики

- рубли и доллары;
- плавающий курс валют и двусторонний обмен;
- здоровье, сытость и счастье;
- более 40 действий;
- работа, образование, развлечения, популярность и политика;
- жильё, активы и пассивный доход;
- случайные события с двумя вариантами решения;
- президентские выборы с риском поражения и санкций;
- сохранение прогресса в `localStorage`;
- поддержка Telegram Web App SDK.

## Изменения v1.16

- Показатель энергии полностью удалён из интерфейса и игровой логики.
- Действия больше не блокируются из-за нехватки энергии.
- Количество действий ограничивается только игровым временем.
- Когда здоровье, сытость или счастье достигают 0, сразу появляется предупреждение.
- Если показатель непрерывно остаётся на нуле следующие 24 игровых часа, персонаж умирает.
- Поднятие показателя выше 0 полностью сбрасывает его смертельный таймер.
- Старые сохранения автоматически очищаются от показателя энергии.

## Запуск на Windows

1. Полностью распакуйте папку `homeless_survival_tg_app_v1_16`.
2. Запустите `start_game.bat`.
3. Игра откроется напрямую в браузере.

Для запуска через локальный HTTP-сервер используйте `start_server.bat`. Для остановки сервера — `stop_game.bat`.

## Публикация на GitHub Pages

1. Загрузите `index.html`, `styles.css` и `app.js` в корень репозитория.
2. Откройте `Settings → Pages`.
3. Выберите `Deploy from a branch`.
4. Выберите ветку `main` и папку `/(root)`.
5. Используйте выданную HTTPS-ссылку в настройках Telegram Web App.

Сохранение хранится в браузере конкретного устройства. Для общего прогресса между устройствами потребуется сервер и база данных.
