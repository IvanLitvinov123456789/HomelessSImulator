# Симулятор выживания бомжа — v2.9.1

Изменения версии:

- работа снова не требует рублей за выполнение действия;
- в карточках работы показывается только возможный доход;
- постоянная рабочая смена также выполняется без денежной платы;
- работа по-прежнему расходует время, здоровье, сытость и счастье;
- правило платных действий в еде, здоровье, развлечениях, учёбе, медиа и политике сохранено;
- лимит обмена 100 000 000 долларов сохранён;
- сохранения v2.9 совместимы.

Запуск: распакуйте архив и откройте `start_game.bat`.
