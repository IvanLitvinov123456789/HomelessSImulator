# Симулятор выживания бомжа — v2.3

Изменения:
- во вкладке «Карьера» сначала показана текущая ступень, затем текущая работа;
- текущая работа первой отображается в категории «Работа» и скрывается при отсутствии работы;
- требования глав на главной и во вкладке «Карьера» синхронизированы;
- все пороги показателей сравниваются с округлёнными значениями, показанными игроку;
- при смене категории действий страница возвращается к началу списка;
- бизнесы стали прибыльнее, улучшения дешевле;
- работа больше не повышает харизму: развивает силу, интеллект, предпринимательство или политику;
- случайные события показывают шанс и дают заметные хорошие или плохие последствия.
