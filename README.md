# Симулятор выживания бомжа — v2.8.1

Версия v2.8.1 основана на v2.8.

## Изменения

- В пошаговое обучение добавлен отдельный рассказ о каждом разделе меню действий.
- Последовательно показываются: «Еда», «Работа», «Здоровье», «Развлечения», «Учёба», «Популярность» и «Политика».
- Для каждого раздела объясняется его назначение, основные награды и возможные затраты.
- Во время обучения нужная категория автоматически выбирается и прокручивается в центр строки категорий.
- Сохранена возможность пропустить обучение или запустить его заново через справку.
- Остальные механики и баланс v2.8 не менялись.
- Сохранения предыдущих версий совместимы.

## Запуск

Запустите `start_game.bat` или загрузите `index.html`, `styles.css`, `app.js` и `.nojekyll` в корень GitHub Pages.
