# Симулятор выживания бомжа — v2.8.2

Версия v2.8.2 основана на v2.8.1.

## Изменения

- Исправлено дрожание и небольшое смещение экрана при частых тапах на телефоне.
- В Telegram Mini App отключён системный вертикальный свайп, который мог тянуть приложение вниз.
- Заблокирован резиновый overscroll у верхней и нижней границы страницы.
- Для кнопок и кликабельных карточек отключены двойное масштабирование, выделение текста и системное меню долгого нажатия.
- На сенсорных устройствах убрана анимация уменьшения карточки при каждом тапе.
- При полном перерисовывании интерфейса сохраняется текущая позиция прокрутки.
- Обычная вертикальная прокрутка и горизонтальная прокрутка категорий сохранены.

## Запуск

Распакуйте архив и запустите `start_game.bat`.

Для GitHub Pages загрузите содержимое папки версии в корень репозитория.
